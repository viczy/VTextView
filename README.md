# VTextView

###Installation

`pod 'VTextView', :git => 'https://github.com/viczy/VTextView.git'`

###Demo

`git clon https://github.com/viczy/VTextView.git`

###Requirements

* iOS 5.0+
* Xcode 6.1

###Description
该控件是自定义的TextView，为了实现类似于微博的textview效果，控件是基于[egotextview](https://github.com/enormego/EGOTextView)做了修改。
